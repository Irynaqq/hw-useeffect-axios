# React + useEffect + axios

Навчальний проєкт на React, створений за допомогою Vite.  
Демонструє виконання HTTP-запитів за допомогою бібліотеки **axios** та хука **useEffect**, а також обробку станів завантаження, помилок та успішного отримання даних.

## Мета

- Освоїти базову роботу з `useEffect` та асинхронними функціями.
- Навчитися використовувати `axios` для HTTP-запитів.
- Реалізувати обробку станів: `loading`, `error`, `data`.
- (Опціонально) Реалізувати рефетч даних при зміні параметра `id`.

## Основні компоненти

- `DataFetcher.jsx` — компонент, який:
  - виконує GET-запит до API (`https://jsonplaceholder.typicode.com/posts/:id`);
  - використовує `useEffect` + `async/await`;
  - показує стани завантаження, помилки та дані;
  - дозволяє переключати `postId` і перевантажувати дані (рефетч).
